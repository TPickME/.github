## 🌨️ О нас

**TPickME** — это коллекция инструментов, рождённых в зимних вечерах за настройкой Arch Linux.  
Здесь ты не найдёшь воды, только чистый код, свободу и уют киберпанка.

Наша цель — дать людям **простой доступ** к сложным вещам:
- 🛡️ Обход блокировок
- 🎨 Редакторы кода
- 📚 Обучение без скуки
